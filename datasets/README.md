# Datasets

This folder contains the datasets used for training the ScamGuard AI scam/phishing detection model.

## Data Sources

### 1. HuggingFace: `ealvaradob/phishing-dataset` (SMS subset)
- **What:** 5,971 SMS messages including spam, smishing (SMS phishing), and legitimate (ham) messages
- **Source:** [HuggingFace Hub](https://huggingface.co/datasets/ealvaradob/phishing-dataset)
- **Original Research:** Mishra, Sandhya & Soni, Devpriya (2022) — Mendeley Data
- **Labels:** 1 = Phishing/Scam, 0 = Legitimate

### 2. Kaggle: Adversarial Scam Dataset
- **What:** ~1,200 manually labeled scam messages (financial fraud, recruitment scams, phishing)
- **Source:** [Kaggle](https://www.kaggle.com/datasets/txnguyen292/adversarial-scam-dataset)
- **Published:** IEEE International Conference on Big Data, 2024
- **Labels:** Original scam, adversarial scam (LLM-modified to evade detection), non-scam

## Files

| File | Description | Generated By |
|------|-------------|-------------|
| `original/merged_scam_dataset.csv` | Raw merged dataset from both sources | `01_data_download.py` |
| `processed/cleaned_scam_dataset.csv` | Text-cleaned version (lowercase, no URLs/punctuation) | `02_data_preprocessing.py` |
| `processed/X_tfidf.pkl` | TF-IDF feature matrix (sparse) | `03_feature_extraction.py` |
| `processed/X_finbert.pkl` | FinBERT [CLS] embeddings (dense) | `03_feature_extraction.py` |
| `processed/y_labels.pkl` | Label array (0=legitimate, 1=scam) | `03_feature_extraction.py` |

## Regenerating
